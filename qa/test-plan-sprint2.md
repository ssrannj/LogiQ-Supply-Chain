# QA Test Plan - Sprint 2

This document outlines the scope for validation support in Sprint 2.

## Scope

- **Catalog**: Verify product listings, categorization, filtering, and search functionalities.
- **Auth**: Test user registration, login, logout, password reset, and session management.
- **Wishlist**: Validate adding/removing items from the wishlist, and wishlist persistence across sessions.
- **Checkout Upload**: Test the functionality of uploading necessary documents or assets during the checkout process.
- **Tracking**: Verify end-to-end order tracking updates and status accuracy.
- **Warranty**: Check warranty claim forms, status updates, and policy enforcement mechanics.
